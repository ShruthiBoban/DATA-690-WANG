# Basic Syntax

## Heading - # H1, ## H2, ### H3
## Bold - **bold text**
## Italic - *italicized text*
## Blockquote - > blockquote
## Ordered List - 1. First Item, 2. Second Item, 3. Third Item
## Uordered List -  - First Item, - Second Item, - Third Item
## Code - `code`
## Horizontal Rule -  ---
## Link - [title](https://www.example.com)
## Image - ![alt text](image.jpg)
